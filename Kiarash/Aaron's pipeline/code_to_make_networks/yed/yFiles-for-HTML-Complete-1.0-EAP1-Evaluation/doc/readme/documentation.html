yFiles for HTML Demos and Documentation

A number of resources are included to help you learn more about using yFiles for HTML:

Included Resources

Source Code Demos

The demos/ directory contains extensive tutorial source code demo applications.

Getting Started Information

In the Getting Started chapter of the yFiles for HTML Developer's Guide: UI Components, you will find information on how to setup your development environment.

Developer's Guides

The doc/api/dguide/ directory contains the more advanced, comprehensive Developer's Guides which describe the architecture and the main concepts used in yFiles:

  • yFiles for HTML Developer's Guide: UI Components
  • yFiles for HTML Developer's Guide: Algorithms and Layout

API Documentation

The doc/api/ directory contains API documentation of the yFiles for HTML library.
