yFiles for HTML in your Project

Library Files

The yFiles for HTML library consists of the Javascript files in the lib/yfiles/ directory and the demos/resources/style/graphclient.css CSS file.
The license file demos/resources/license.js needs to be included in any HTML page that uses the library.

For detailed descriptions see also the Setup your Development Environment section of the yFiles for HTML Developer's Guide.

Documentation

To get quick access to the API documentation in your IDE, you'll want to add the API documentation in the doc/api/html/ directory to your project.
Auto-completion support in your IDE can be enabled through the ide-support/yfiles-api.js file.

For detailed descriptions see also the Setup your Development Environment section of the yFiles for HTML Developer's Guide.
