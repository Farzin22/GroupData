yFiles for HTML Technical Requirements

Prerequisites

Development

We suggest to use a modern IDE, e.g., IntelliJ WebStorm, IntelliJ IDEA, Netbeans, or Eclipse.

Deployment

To run yFiles for HTML applications, including the provided demo applications, you will need to have an HTML5-capable browser, e.g. one of the following:

  • Internet Explorer 9
  • Firefox 12
  • Chrome 19
  • Safari 5.1
  • Opera 11.61
