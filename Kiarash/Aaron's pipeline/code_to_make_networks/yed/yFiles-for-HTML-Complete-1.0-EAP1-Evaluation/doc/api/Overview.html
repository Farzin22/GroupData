yFiles for HTML 1.0 README

yFiles for HTML is an extensive HTML5/Javascript class library. It provides diagram visualization controls, data structures and algorithms for graph analysis, and advanced functionality to automatically calculate layouts for graphs, diagrams, and networks.

Documentation

The following documentation for yFiles for HTML 1.0 is available:

  • API documentation (Class Reference)
  • Developer's Guide (UI Components, Analysis and Layout)
  • programming samples descriptions
  • Software License Agreement for yFiles for HTML; you have to agree to these terms before using yFiles for HTML

Programming Samples

yFiles for HTML comes with many programming samples. These include full source code that presents the functionality of the yFiles for HTML diagramming library.

Getting Started

The yFiles for HTML documentation and the programming samples are the primary source of information.

The Developer's Guide (UI Components, Analysis and Layout) introduces the concepts and presents the library's range of features together with any types involved. It has many links into the API documentation (Class Reference) so that you can quickly find all relevant information about classes, interfaces, methods, properties, etc. Also, it provides pointers to relevant source code examples from the programming samples that show discussed features.

The Developer's Guide is your indispensable companion when working with the yFiles for HTML diagramming library.
