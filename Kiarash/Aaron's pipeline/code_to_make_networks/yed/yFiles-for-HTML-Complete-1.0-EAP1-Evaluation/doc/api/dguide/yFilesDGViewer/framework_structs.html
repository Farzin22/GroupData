Structs

Defining a Struct

Structs are value types. They have the same definition as classes, except that they always directly extend yfiles.lang.Struct. To create a new struct type, use yfiles.lang.Struct as a function.

Example 3.15. Struct definition

var RectD = yfiles.lang.Struct('RectD', {
  $with: [IRectangle, ICloneable],
  $topLeft: null,
  width: 0, height: 0,

  constructor: function(x, y, w, h) {
      this.$topLeft = new PointD(x, y);
      this.width = w;
      this.height = h;
  },

  get topLeft() {
      return this.$topLeft.clone();
  }
});

Structs are defined just like classes. They have the same semantics and work the same way with one major difference:

If a class or struct is cloned using memberwiseClone, all members that are structs are cloned as well.

In the above example, $topLeft is a PointD, which is a struct. If we clone an instance of RectD, then the copy will have a cloned $topLeft member. Note that while structs have a built-in clone method, you are responsible for using it to implement struct semantics. That is, you must clone structs before passing them to a method or returning them from one.

Structs also have more specialized versions of the equals and hashCode methods. Two struct instances with the same values are equal and have the same hash code. This differs from the yfiles.lang.Object definitions for equals and hashCode, which ensure that only identical objects are the same.

Example 3.16. The createDefault method

var rect = RectD.createDefault();

All struct types have a createDefault method which creates a new, uninitialized instance of the struct without calling any of its constructors. This is useful when you want to create the variable earlier in your code, and instantiate it later.

Lazy Structs

A lazy struct has the same properties as a lazy class. It must be namespaced to a yfiles.module and is evaluated at first use.

Example 3.17. Defining a lazy struct

yfiles.module('myapp.model', function(exports) {
  exports.Name = new yfiles.StructDefinition(function() {
      return {
          firstName: "", lastName: "",

          constructor: function(firstName, lastName) {
              this.firstName = firstName;
              this.lastName = lastName;
          },

          toString: function() {
              return this.firstName + " " + this.lastName;
          }
      };
  });
});
