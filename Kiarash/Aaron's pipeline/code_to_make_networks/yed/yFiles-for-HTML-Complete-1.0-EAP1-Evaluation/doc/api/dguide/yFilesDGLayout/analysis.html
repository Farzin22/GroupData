This chapter presents the graph analysis algorithms provided by the yFiles for HTML library and explains how to use them.

Using yFiles Graph Analysis Functionality

Internally, the analysis algorithms in the yFiles Basic library component are tailored to the basic algorithms graph structure. By means of the adapter mechanisms presented in Chapter 3, Using yFiles for HTML Algorithms Functionality, however, the analysis algorithms can be used with an IGraph-based graph structure as well.

Adapter class YGraphAdapter provides services for necessary conversions prior and subsequent to running an analysis algorithm. Most notably, it creates a copy of an IGraph that can be used with such algorithms:

Example 4.1. Basic preparation for calling an analysis algorithm

// 'graph' is of type yWorks.yFiles.UI.Model.IGraph.

// Create the graph model adapter to get a proper analysis graph structure.
var /*yworks.yfiles.ui.model.YGraphAdapter*/ graphAdapter = new yworks.yfiles.ui.model.YGraphAdapter(this.$graph$0);

// The YGraph property returns the yWorks.yFiles.Algorithms.Graph instance that
// corresponds to the given IGraph above.
var /*boolean*/ isConnected = yworks.yfiles.algorithms.GraphChecker.isConnected(graphAdapter.yGraph);

The following detailed example shows the invocation of an analysis algorithm with additional intermittent conversions and updates to the original IGraph.

Example 4.2. Graph analysis with an IGraph-based graph using adapter mechanisms

// 'graph' is of type yWorks.yFiles.UI.Model.IGraph.

// Create the graph model adapter to get a proper analysis graph structure.
var /*yworks.yfiles.ui.model.YGraphAdapter*/ graphAdapter = new yworks.yfiles.ui.model.YGraphAdapter(this.$graph$0);

// Test if the graph is a tree.
if (yworks.yfiles.algorithms.Trees.isTree(graphAdapter.yGraph)) {
  // Explicitly set the first node from the node set as the root node of the
  // tree.
  // Direct the tree; i.e., revert those edges that point towards the root
  // node.
  var /*yworks.yfiles.algorithms.EdgeList*/ el = yworks.yfiles.algorithms.Trees.directTreeWithRoot(graphAdapter.yGraph, graphAdapter.getCopiedNode(yfiles.system.collections.EnumerableExtensions./*<yworks.yfiles.ui.model.INode>*/first(this.$graph$0.nodes)));
  // Transfer back the result.
  var /*yfiles.system.collections.generic.IEnumerable`1<yworks.yfiles.ui.model.IEdge>*/ el_igraph = graphAdapter.createEdgeEnumerable(el);
  var /*yfiles.util.IEnumerator*/ tmpEnumerator = el_igraph.getEnumerator();
  while (tmpEnumerator.moveNext()) {
    var /*yworks.yfiles.ui.model.IEdge*/ e = tmpEnumerator.current;
    {
      // Reverse the corresponding edges in the original IGraph.
      yworks.yfiles.ui.model.GraphExtensions.reverse(this.$graph$0, e);
    }
  }

  // Get the leaf nodes of the tree.
  var /*yworks.yfiles.algorithms.NodeList*/ nl = yworks.yfiles.algorithms.Trees.getLeafNodes(graphAdapter.yGraph, true);
  var /*yfiles.system.collections.generic.IEnumerable`1<yworks.yfiles.ui.model.INode>*/ nl_igraph = graphAdapter.createNodeEnumerable(nl);
}
