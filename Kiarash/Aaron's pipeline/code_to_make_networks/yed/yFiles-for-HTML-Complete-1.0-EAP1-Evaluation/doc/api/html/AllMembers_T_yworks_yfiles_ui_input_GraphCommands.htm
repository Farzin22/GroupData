The GraphCommands type exposes the following members.

Fields

  Name Description
ADD_LABEL_COMMAND
A routed UICommand that can be executed from within the GraphControl to add another label to the ILabeledItemthat is specified by the parameter.
ADJUST_GROUP_NODE_SIZE_COMMAND
A routed UICommand that can be executed from within the GraphControl to adjust the size of the selected group nodes.
BEGIN_EDGE_CREATION_COMMAND
This is an RoutedUICommand that instances CreateEdgeInputMode instances can satisfy requests for if they have been installed in a CanvasControl.
COLLAPSE_GROUP_COMMAND
A routed UICommand that can be executed from within the GraphControl to collapse the currently selected group nodes.
DESELECT_ALL_COMMAND
A routed UICommand that can be executed from within the GraphControl to clear the selection.
DESELECT_ITEM_COMMAND
A routed UICommand that can be executed from within the GraphControl to deselect the item that is specified by the parameter.
EDIT_LABEL_COMMAND
A routed UICommand that can be executed from within the GraphControl to edit the label that is specified by the parameter.
ENTER_GROUP_COMMAND
A routed UICommand that can be executed from within the GraphControl to switch the local root of the current IFoldedGraph view to the given group node.
EXIT_GROUP_COMMAND
A routed UICommand that can be executed from within the GraphControl to exit the current local root of the current IFoldedGraph view to the given parent view.
EXPAND_GROUP_COMMAND
A routed UICommand that can be executed from within the GraphControl to expand the currently selected group nodes.
GROUP_SELECTION_COMMAND
A routed UICommand that can be executed from within the GraphControl to group the currently selected items.
SELECT_ITEM_COMMAND
A routed UICommand that can be executed from within the GraphControl to select the item that is specified by the parameter.
TOGGLE_GROUP_STATE_COMMAND
A routed UICommand that can be executed from within the GraphControl to toggle the collapsed/expanded state of currently selected group nodes.
TOGGLE_ITEM_SELECTION_COMMAND
A routed UICommand that can be executed from within the GraphControl to toggle the selection state of the item that is specified by the parameter.
UNGROUP_SELECTION_COMMAND
A routed UICommand that can be executed from within the GraphControl to ungroup the currently selected items.
