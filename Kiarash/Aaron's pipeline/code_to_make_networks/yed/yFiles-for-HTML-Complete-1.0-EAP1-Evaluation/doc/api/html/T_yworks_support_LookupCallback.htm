LookupCallback Delegate
See Also Send Feedback
Callback delegate that can be used to query results of a virtual lookup(Class) call on objects that don't necessarily implement ILookup.

Namespace: yworks.support

Syntax

JavaScript
delegate :Object LookupCallback(
	subject:Object,
	type:Class
)

Parameters

subject
Type: yfiles.lang..::..Object
the subject for which a lookup shall be made
type
Type: yfiles.lang..::..Class
the type to look up

Return Value

an instance that can be assigned to type or null
