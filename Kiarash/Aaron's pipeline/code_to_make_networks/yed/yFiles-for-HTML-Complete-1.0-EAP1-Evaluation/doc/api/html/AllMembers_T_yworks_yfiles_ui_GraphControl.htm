dragSize
Gets or sets the area in view coordinates the mouse may stay in before a movement is considered a drag. If the mouse is moved within this area longer than dragTime milliseconds, the movement will be considered a drag, nevertheless. The larger the area the later a mouse movement will be recognized as a drag. This influences the click-sensitivity, since a mouse button release is only considered a click if there was no drag since the last mouse button press. The default value is (5, 5).
(Inherited from CanvasControl.)
dragTime
Gets or sets the value of the drag time. This value indicates the amount of time that may pass before a mouse movement is considered a drag when the mouse stays within its dragSize area. The higher the value the later a mouse movement will be recognized as a drag. This influences the click-sensitivity, since a mouse button release is only considered a click if there was no drag since the last mouse button press. The default value is 300 milliseconds.
(Inherited from CanvasControl.)
