BridgeManager Class
Members See Also Send Feedback
Helper managing class that can be used in conjunction with canvasControl instances to manage the calculation of bridges for paths over obstacles.

Namespace: yworks.canvas.model

Syntax

JavaScript
class BridgeManager

Remarks

This class will typically be used to prepare the IRenderContext for rendering in a CanvasControl. Setting the canvasControl property to a non-nullNothingnullptrnull value will register this class for the preparation of the context. It will make itself available via the lookup(Class) mechanism in the IRenderContext. During the creation of the visuals, this instance can be queried and the addBridges(IRenderContext, GeneralPath, IAddBridgesCallback) method can be used to calculate bridges for a GeneralPath. Initially IObstacleProvider instances need to be registered with this instance via the addObstacleProvider(IObstacleProvider) instance so that crossings can be later determined between the path and the obstacles.
