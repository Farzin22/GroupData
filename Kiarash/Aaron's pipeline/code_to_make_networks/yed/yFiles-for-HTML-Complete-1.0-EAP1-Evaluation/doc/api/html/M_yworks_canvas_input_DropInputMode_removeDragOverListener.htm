DropInputMode Class See Also Send Feedback
This event is raised, if a drag operation drags over the CanvasControl into which this instance is installed.

Namespace: yworks.canvas.input

	$dragOver$2Event:function
)

Parameters

$dragOver$2Event
Type: function
The listener to remove.
function has the signature function(sender:Object, e:TEventArgs) as defined in the delegate function.

Remarks

No event is raised
  • if this instance is not enabled,
  • if another concurrent input mode is active or
  • the data from the drag operation does not conform to the expected format.
To get additional data for the event, clients should use the sender which is set to the sending DropInputMode. The EventArgs provide no useful data.
