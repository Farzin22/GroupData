Properties

  Name Description
allowMirroring
Specifies whether or not subgraph components may be mirrored to improve the layout quality, i.e., for each component the algorithm checks which of the four possible mirrorings minimizes the edge length.
componentAssignmentStrategy
The strategy that is used to assign partial nodes to subgraph components.
considerNodeAlignment
True, if node alignment is enabled, that is the algorithm tries to align partial nodes with other nodes.
coreLayouter
The core layouter, i.e., the layout algorithm that is applied to the subgraph components.
(Overrides AbstractLayoutStage..::..coreLayouter.)
edgeRouter
The edge router used for routing partial edges as well as edges between different subgraph components (so-called inter-edges).
edgeRoutingStrategy
The specified edge routing strategy.
fixedGroupResizingEnabled
Specifies whether or not fixed group nodes may be resized.
layoutOrientation
The layout orientation.
maximalDuration
The time limit (in milliseconds) set for the layout algorithm.
minimalNodeDistance
The minimum distance between two adjacent nodes.
optimizeLayoutOrientation
Specifies whether or not a postprocessing step should be applied to reduce the number of edges that do not comply with the desired orientation.
packComponents
If enabled, the bounding boxes of subgraph components may overlap (elements of different components will still not overlap).

The objective used for finding 'good' positions for subgraph components.
routeInterEdgesImmediately
Specifies whether or not edges between different subgraph components should be routed immediately.
