This demo shows how to use the NodeDropInputMode" class for drag and drop. In contrast to DropInputMode, NodeDropInputMode shows a preview of the node while dragging and uses snapping.

Things to Try

  • To create a node, drag the desired node style from the left panel onto the canvas.
  • Drag the node around on the canvas to see grid snapping.
  • Drag a node around near another node to see node to node snapping.
  • Drag a node over a group node to see group node highlighting.
  • Drop a node over a group node to place it inside the group.
  • Switch off snapping or snapping and preview using the combo box above the canvas.
