This demo shows how to create collapse/expand functionality for trees with the help of the FilteredGraphWrapper class for filtering subtrees, and the TreeLayouter class for the actual layout.

Things to Try

  • Click a node with a +/- button to expand or collapse its subtree. Note that there are also leaf nodes which cannot be expanded further.
  • Select an automatic layout style from the dropdown in the toolbar. Choosing a new style, or expanding and collapsing nodes, triggers a refresh of the layout.
  • Use the zoom and "Fit Content" buttons in the toolbar to focus on local areas or to view the complete graph.
