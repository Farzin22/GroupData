yFiles for HTML Programming Samples and Tutorials README

The yFiles for HTML programming samples and tutorials are extensive sample applications with HTML/JavaScript source that present the functionality of the yFiles for HTML diagramming library.

Running a Sample

To execute a specific programming sample, either just follow the corresponding link from the table below, or open the file index.html in the specific demo subdirectory. It is also possible deploy the samples to a web server and load the samples from there.

Due to strict cross domain policies in both Internet Explorer and Google Chrome, many demos work only partially or not at all when viewed from a filesystem location in these browsers. The preferred workaround is to access the samples through a web server instead. Google chrome can also be instructed to use a more tolerant policy by starting chrome.exe with the --allow-file-access-from-files command line option. Also, Firefox has no such problems.

Available demos

The following table lists the relevant files and subdirectories within your yFiles for HTML programming samples and tutorials installation directory.

Name Description
demo.yfiles.algorithms.shortestpath Shows how to use graph analysis algorithms in yFiles for HTML, specifically, shortest path search, together with the YGraphAdapter analysis adapter class.
demo.yfiles.graph.collapse Shows how to create collapse/expand functionality for trees with the help of the FilteredGraphWrapper class for filtering subtrees, and the TreeLayouter class for the actual layout.
demo.yfiles.graph.incrementalhierarchicgrouping Shows how to expand and collapse sub graphs organized in groups and folders.
demo.yfiles.graph.input.draganddrop Shows how to use the NodeDropInputMode" class for drag and drop
demo.yfiles.graph.orgchart Complex organization chart example.
demo.yfiles.graph.simplecustomstyle Shows how to create custom styles for nodes, edges, labels, ports, and edge arrows from scratch and how to use them.
demo.yfiles.graph.simpleeditor Simple editor application.
demo.yfiles.graph.tableeditor Shows how to interactively modify a Table.
demo.yfiles.graph.viewer A simple graph viewer application.
demo.yfiles.io.fileoperation Shows various ways to open from and save to a GraphML file.
demo.yfiles.toolkits.dojo.simple Shows how to integrate yFiles for Html with Dojo 1.7 and later.
demo.yfiles.toolkits.jquery.simple Shows how to integrate yFiles for Html with jQuery.
